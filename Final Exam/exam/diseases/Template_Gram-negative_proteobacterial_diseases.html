How to manage this template's initial visibility
To manage this template's visibility when it first appears, add the parameter:

|state=collapsed to show the template in its collapsed state, i.e. hidden apart from its titlebar  e.g. {{Gram-negative proteobacterial diseases |state=collapsed}}
|state=expanded to show the template in its expanded state, i.e. fully visible  e.g. {{Gram-negative proteobacterial diseases |state=expanded}}
|state=autocollapse to show the template in its collapsed state but only if there is another template of the same type on the page  e.g. {{Gram-negative proteobacterial diseases |state=autocollapse}}

Unless set otherwise (see the |state= parameter in the template's code), the template's default state is autocollapse.
