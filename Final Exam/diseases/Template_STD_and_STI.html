Template documentation

How to manage this template's initial visibility
To manage this template's visibility when it first appears, add the parameter:

|state=collapsed to show the template in its collapsed state, i.e. hidden apart from its titlebar  e.g. {{STD and STI |state=collapsed}}
|state=expanded to show the template in its expanded state, i.e. fully visible  e.g. {{STD and STI |state=expanded}}
|state=autocollapse to show the template in its collapsed state but only if there is another template of the same type on the page  e.g. {{STD and STI |state=autocollapse}}

Unless set otherwise (see the |state= parameter in the template's code), the template's default state is autocollapse.

This template is part of the Medical series of navigation boxes. Refer to Template:Medicine and its talk page for suggestions on style and editing.

Editors can experiment in this template's sandbox (create | mirror) and testcases (create) pages.
